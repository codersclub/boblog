Bo-Blog 2.1.1 Инструкция по установке.

1、Инсталляция:
Для установки Bo-Blog 2.1.1，сервер （или виртуальный хост）должен удовлетворять следующим условиям:：
*PHP версии 4.1.0  и выше；
*Поддержка MySQL 4.0.0 и выше；
*Поддержка сессий и куков；
*Не является обязательным для добавления рекламы и ограничений в выводе страниц

Для нормальной работы Bo-Blog 2.1.1, Ваш сервер должен отвечать следующим условиям:
*PHP версии 4.1.0  и выше；
*Установленная GD библиотека；
*Установлена Zlib Function (работа со сжатыми файлами gzip (.gz) )；

Если отключены следующие функции, не будет работать должным образом 
*opendir / readdir
*unlink
*fopen / fsockopen
*error_reporting
*chdir

2、Новые  установки：
2.1. Установите права 777 на следующие папки Bo-Blog, загруженные на сервер Unix：
  attachment/
  bak/
  data/
  post/
  temp/
  temp/openid/
  temp/openid/associations/
  temp/openid/nonces/
  temp/openid/temp/


2.2. Запуск установки / install.php. Следуйте инструкциям, чтобы завершить установку.
2.3. Если ваш сервер поддерживает Gzip, в параметрах блога, разрешите сжатие Gzip.
2.4. При инсталляции выбрать принудительную кодировку UTF-8.


3、Дистрибутив состоит из:
нескольких папок, входящих в установочный пакет：
*bo-blog - сама программа
*documents - Документация
*tools - Некоторые инструменты см. прилагаемый документ
